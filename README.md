# Smart Task Manager

A modern and interactive **Task Management Web Application** built using **HTML, CSS, and JavaScript**.  
The application helps users organize tasks, track progress, and manage productivity through a clean and responsive interface.

Live Demo:  
https://megu2024.github.io/smart-task-manager/

---

## Features

• Add new tasks easily  
• Mark tasks as completed using checkboxes  
• Delete tasks  
• Set task priority (High / Medium / Low)  
• Add task deadlines  
• Filter tasks (All / Completed / Pending)  
• Progress tracking with dynamic progress bar  
• Dark mode support  
• Persistent storage using browser **Local Storage**

---

## Technologies Used

• HTML5  
• CSS3  
• JavaScript (ES6)  
• Local Storage API

---

## User Interface

The application provides a clean and intuitive UI with:

• Gradient modern background  
• Responsive task cards  
• Priority indicators  
• Interactive checkboxes  
• Real-time progress tracking
