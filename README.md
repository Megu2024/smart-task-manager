# Smart Task Manager

A modern and interactive **Task Management Web Application** built using **HTML, CSS, and JavaScript**.
The application allows users to efficiently manage their daily tasks with features like task prioritization, deadlines, progress tracking, and dark mode.

## Live Demo

🔗 https://megu2024.github.io/smart-task-manager/

---

## Features

* Add new tasks easily
* Mark tasks as completed using checkboxes
* Delete tasks
* Set task priority (High / Medium / Low)
* Add task deadlines
* Filter tasks (All / Completed / Pending)
* Progress tracking using a dynamic progress bar
* Dark mode toggle
* Data persistence using browser **Local Storage**

---

## Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive UI
* **JavaScript (ES6)** – Application logic and interactivity
* **Local Storage API** – Persistent task storage in the browser

## How It Works

1. The user enters a task in the input field.
2. Tasks can optionally include **priority level** and **deadline**.
3. Tasks are stored in the browser using **Local Storage**.
4. Users can mark tasks as completed using checkboxes.
5. The progress bar automatically updates based on completed tasks.
6. Tasks can be filtered, deleted, or cleared.
